# Toss Submission Checklist

## 제품 포지셔닝

- 서비스 정의를 `스테이블코인 상태 모니터링 도구`로 고정
- 투자, 추천, 수익, 매수/매도 표현 제거
- 정보형·알림형 서비스로만 설명

## 화면 검수 체크

- 홈 첫 화면에서 상태판과 핵심 알림 포인트가 바로 보이는지 확인
- 비교 화면에서 `더 안전` 같은 우열 표현이 없는지 확인
- 설명 카드가 초보 사용자도 이해 가능한 문장인지 확인
- 디스클레이머와 데이터 출처, 업데이트 시각이 항상 노출되는지 확인

## 데이터 체크

- 가격 출처 표기: CoinGecko API
- 국내 가격 출처 표기: Upbit 시세
- 환율 출처 표기: Frankfurter
- 지연/오류 가능성 문구 유지

## 금지 기능

- 거래소 연결
- 매수/매도 버튼
- 지갑 생성/보관
- 송금 기능
- 수익 추천

## 배포 전 환경변수

- `BASE_URL`
- `BASE_MINIAPP_HEADER`
- `BASE_MINIAPP_PAYLOAD`
- `BASE_MINIAPP_SIGNATURE`

## 최종 확인

- `.well-known/farcaster.json` 응답 확인
- 실데이터 응답 `/api/market-snapshot` 확인
- 모바일 화면에서 첫 진입 정보량 과도 여부 확인
- 법무/컴플라이언스 최종 검토
