# Oh-My-Stablecoin-Alarm Product Scope

## 목표

한국 사용자에게 스테이블코인 관련 핵심 정보를 모바일 친화적으로 제공하는 정보형·알림형 미니앱을 만든다.

## 포함 기능

- 스테이블코인별 가격, 기준 대비 변동, 국내외 가격 괴리, 준비금 보고 리듬, 체인 분포 표시
- 사용자가 직접 설정하는 임계치 기반 알림 규칙
- 규제 변화, 공시 지연, 가격 변동 확대를 상태 카드로 요약
- 교육성 설명과 상태 해설
- 국내외 가격 괴리(김치프리미엄)를 투자 권유 없이 참고 지표로 표시

## 제외 기능

- 매수, 매도, 교환, 송금, 예치, 보관
- 주문 라우팅, 체결 유도, 투자 권유
- 지갑 연결을 통한 자산 접근
- 특정 자산 추천이나 수익률 홍보

## 제품 원칙

- "지금 사세요" 대신 "지금 무엇이 변했는가"를 보여준다.
- 수익률이 아니라 가격 변동, 투명성, 공시 주기, 체인 상태를 전면에 둔다.
- 개인 자산을 다루지 않고, 시장 모니터와 알림 허브 역할만 수행한다.
- 토스 검수 기준을 고려해 "추천"과 "안전" 대신 "변동", "상태", "참고 정보"를 사용한다.

## SDLC 핸드오프 체크리스트

모든 영역은 “정보형·알림형” 경계와 한국 규제 리스크를 유지한다는 점을 전제로 한다.  

- **Product**: 최종 시나리오(홈, 비교, 알림, 설명 탭) 및 디스클레이머/출처 문구를 명시하고, 대시보드가 거래·중개·지갑·투자 추천을 하지 않는다는 확약을 포함시킨다.  
- **Design**: 토스 검수 기준의 금지 표현 목록(추천, 안전, 수익률)과 상태 레벨(안정/주의/변동 확대/변동 심화) 가이드를 반영한 UI 스펙 및 모바일 대응 크기/타이포를 정리한다.  
- **Data**: 가격/온체인/공시 출처, 업데이트 타임스탬프, 지표(변동 %, 공급 변화 등) 데이터 파이프라인을 문서화하고 지연/오류 가능성을 명시한다.  
- **Compliance**: 디스클레이머, 표시광고법·자본시장법 검토 결과, 개인정보영향 최소화(로그인·알림 처리 계획 없음)와 관련법 위반 방지 대응을 기록한다.  
- **Engineering**: 정보만 제공하는 스토리보드로 API 호출, 알림 규칙 저장, Base Mini App SDK 브리지, .well-known/manifest, 푸시/알림 시나리오까지 구현 계획을 정리하고, 변경 시 검수 로그를 남긴다.

## 주의

이 문서는 제품 범위를 좁히기 위한 운영 메모이며 법률 자문이 아니다. 실제 출시 전에는 한국 변호사 또는 컴플라이언스 담당자 검토가 필요하다.
